<h1> App Sus </h1>
  <p>To use AppSus, please visit the following link:</p>
  <a href="https://ilansuli.github.io/AppSus/#/">https://ilansuli.github.io/AppSus/#/</a>
<p>Check out Appsus, a suite of web applications inspired by Google's popular services, Gmail and Keep.
 Each app has it's full CRUDL support, a responsive design and with a special feature to integrate between the apps, transforming an email to a note.
The applications were built using Vue.js, a versatile and easy-to-use JavaScript framework for building user interfaces.</p>
  <p>This project was created in collaboration with Lior Doron.</p>

  <h2>Technologies used</h2>
  <ul>
 <li>Vue.js</li>
 <li>CSS</li>
 <li>JavaScript</li>
  </ul>
